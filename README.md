# 快件末端驿站综合调度与运营桌面系统

基于 Electron + Vue 3 + Element Plus 构建的快递驿站综合管理桌面应用。

## 功能模块

### 1. 首页概览 (Dashboard)
- 今日入库、出库、在库、超时滞留统计
- 近7天出入库趋势图
- 快递公司入库分布饼图
- 格口使用情况柱状图
- 最近入库记录

### 2. 扫描入库 (Scan In)
- 运单号扫描/手动输入
- 智能格口分配算法：根据包裹尺寸、冷藏要求、柜格占用状态自动匹配最佳格口
- 支持普通柜和冷藏柜分类管理
- 6位随机取件码自动生成
- 模拟短信通知推送
- 格口状态实时可视化展示

### 3. 取件核验 (Pickup)
- 支持输入6位取件码或扫码核验
- 连续3次错误自动锁定并生成异常工单
- 管理员解锁功能
- 超48小时未取自动计算保管费（每天1元）
- 取件成功后格口自动释放
- 取件记录实时追踪

### 4. 运单管理 (Orders)
- 运单列表查询（运单号/收件人/手机号/状态/日期）
- 新增、编辑运单信息
- 包裹锁定/解锁
- 退件处理
- 分页展示

### 5. 格口管理 (Lockers)
- 总格口数、空闲数、占用数、占用率统计
- 按区域、类型、尺寸、状态筛选
- 格口可视化状态展示
- 新增、编辑格口配置
- 强制释放占用格口

### 6. 退件管理 (Return)
- 用户在线提交退货申请
- 退件原因、取件地址、期望时间窗口录入
- 系统自动生成退运单号（RT前缀）
- 快递员分配与上门取件时间窗口
- 退件状态流转：待取件→已取件→已寄出→已完成
- 退件费自动结算（默认8元）

### 7. 财务结算 (Finance)
- 按快递公司自动汇总派费、超时保管费、退件费
- 派费收入、保管费收入、退件费收入分类统计
- 支持按日期范围筛选
- 对账结算单PDF导出
- 批量结算功能
- 收支明细记录查询

### 8. 统计报表 (Reports)
- 日报/月报切换
- 核心指标：入库量、出库量、滞留件数、超时率、滞留率、平均取件时长、营收
- 出入库趋势图（柱状图）
- 快递公司业务占比（饼图）
- 包裹尺寸分布图
- 超时率与滞留率对比折线图
- 快递公司详细统计表（含合计行）
- 月度运营分析报表PDF导出

### 9. 异常工单 (Work Orders)
- 工单类型：取件解锁、包裹破损、投诉建议、设备故障、其他异常
- 状态流转：待处理→处理中→已完成
- 关联单号与问题描述
- 处理人分配与处理时间记录

### 10. 用户管理 (Users)
- 三种角色权限：管理员(admin)、快递员(courier)、普通用户(user)
- 基于角色的菜单权限控制
- 用户新增、编辑、删除
- 密码重置（默认123456）

## 技术栈

- **前端框架**: Vue 3 (Composition API)
- **UI组件库**: Element Plus 2.x
- **路由**: Vue Router 4
- **状态管理**: Pinia
- **图表**: ECharts 5
- **PDF导出**: jsPDF + jsPDF-autotable
- **桌面容器**: Electron 29
- **本地数据库**: 
  - Electron环境：better-sqlite3 (SQLite)
  - 浏览器环境：localStorage + 内存模拟SQL引擎
- **构建工具**: Vite 5
- **日期处理**: dayjs

## 安装运行

```bash
# 安装依赖
npm install

# 纯前端开发模式（浏览器运行，使用模拟数据库）
npm run dev

# Electron桌面应用开发模式
npm run electron:dev

# 构建生产版本
npm run build

# 打包桌面应用（Windows）
npm run electron:build
```

## 核心业务逻辑说明

### 智能格口分配算法
1. 过滤条件：状态=空闲、类型匹配（冷藏/普通）
2. 尺寸匹配优先级：按与目标尺寸最接近的尺寸排序
3. 区域分布：A区/B区分层管理

### 超时保管费计算
- 入库后48小时内免费
- 超过48小时后，每满24小时收取1元
- 取件核验时自动计算并显示

### 取件码锁定机制
- 连续输入错误取件码3次自动锁定
- 锁定后生成异常工单通知管理员
- 仅管理员可解锁，解锁后错误次数清零

### 财务结算规则
- 派费：每件1.5元，入库时入账
- 超时保管费：取件结算时入账
- 退件费：每件8元，退件完成时入账
